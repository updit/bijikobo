# CLAUDE.md — bijikobo

## プロジェクト概要
美字工房（iljos.com）の公式サイト。日本語固定・独立サイト。
ローカライズ・インバウンド対応のコトバとデザインを提供する事務所の「ひっそりとした再起動宣言」サイト。
updit（updit.co）とは別ドメイン・別プロジェクトで運用。

## 技術スタック
- Next.js（App Router）
- TypeScript
- Tailwind CSS
- Formspree（お問い合わせフォーム）
- Vercel（デプロイ）

## サイト構成（1ページ）
セクション順：
1. `#screen1` — メインビジュアル（キャッチ・サブキャッチ・ポリシー・隠れキャラ）
2. `#stats` — 統計セクション
3. `#services` — サービスセクション
4. `#contact` — お問い合わせフォーム（Formspree）
5. `#screen2` — コンピュータ画面モチーフ（upditへのリンク）

## 言語設計
- 日本語固定（言語切替なし）
- updit（updit.co）への外部リンクあり

## ブランドカラー
- プライム：#800202（臙脂）、#ecebe4（オフホワイト）
- 背景：#ffffff または #ecebe4
- 文字：#000000

## フォント
- 見出し：游明朝 / Hiragino Mincho ProN（セリフ系）
- 本文：Zen Kaku Gothic New（Google Fonts）

## コピー確定済み

### #screen1
- eyebrow：美字工房 ｜ BIJIKOBO
- キャッチ：コトバの**チカラ**。
- サブキャッチ：「読んで伝わる」から、「見て解る」へ。
- ポリシー：長く休んでいた事務所を、ひっそりと再開しました。多言語対応・インバウンド向けのコトバとデザインで、伝わりにくいものを、見える形に変えていきます。
- 隠れキャラ：ページロード/スクロールと連動して「目が覚める」（真っ黒シルエット）

### #stats
- 広告・販促物制作 10年以上
- 日本語・英語ネイティブによる品質チェック
- 小規模ならではの1：1サポート

### #services
- 既存ウェブサイトの多言語化
- インバウンド向けコンテンツ作成
- UI/UX ローカライズレビュー
- 英語圏向けプロモーション

### #contact
- タイトル：お気軽にお問い合わせください。
- フォーム：お名前 / Email / 内容
- 注記：通常、24時間以内にお返事いたします。海外ベースのため、メールを中心としたやりとりになりますことを、ご了承ください。

### #screen2
- コンピュータ画面モチーフ
- 画面内にupditのファーストビューが表示
- クリックでupdit.coへ遷移
